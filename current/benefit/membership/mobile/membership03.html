꼭! 확인하세요.

  • KT 멤버십 고객만 월 1회 응모 가능하며, 응모 후 취소는 불가합니다.
  • 100MB~1GB(1,980원~14,300원 상당, 부가세 포함)의 데이터 혜택이 제공됩니다.
  • LTE 요금제 고객님들을 위한 서비스로 모바일 회선당 월1회 참여 가능합니다. (단, 청소년 요금제/ 선불 요금제 응모 불가)
  • 데이터는 당첨 즉시 응모 회선에 자동 등록되며, 다음달 말일까지 이용 가능합니다.
  • 데이터 차감 순서는 ①LTE 데이터 룰렛 ②이월 데이터 ③당월 데이터입니다. (단, LTE 데이터 룰렛/ LTE 데이터 충전/장기혜택 쿠폰(데이터 1GB)은 등록된 순서대로 차감)
  • 데이터 용량별 당첨 확률은 상이할 수 있습니다.
  • 시스템 점검 시간(매일 23:50~24:10)에는 룰렛 응모가 불가합니다.
