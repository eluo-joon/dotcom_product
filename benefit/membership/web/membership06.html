주요 혜택 및 기능 안내

서비스 이용 안내
KT 패밀리박스는
KT 결합상품 사용 및 서비스에 가입한 고객 대상 App기반의 혜택 제공 서비스로서, 약관에 동의하고 가입한 고객에게 데이터 공유, 제휴 혜택 쿠폰 등 혜택을 제공하는 서비스입니다.
KT 결합 상품 가입자(이하 결합 가족 회원) 및 결합 가족 회원이 초대한 결합 미이용자(이하 일반 가족 회원) 를 대상으로 서비스를 제공합니다.
회원 가입 단위는 모바일 회선 기준으로 이루어 지며, 복수 회선을 가진 경우 각 회선별로 가입이 가능 (단, 모바일 회선은 데이터 전용 요금제를 제외한 휴대전화 요금제에 한함)
용어의 정의
가족 그룹: 동일 결합 그룹에 속한 결합 가족 회원 및 결합 가족 회원이 초대하여 가입된 일반 가족 회원으로 구성된 그룹을 일컬음
결합 가족 회원 : 본 서비스에 가입한 동일 결합 그룹에 속한 결합 가족 회원을 일컬음
일반 가족 회원: 결합 가족 회원이 초대하여 가입한 동일 결합 그룹에 속하지 않은 회원을 일컬음
서비스 이용 대상
KT의 유무선/무선간 결합상품(신규 가입 제한 결합상품 포함) 회원 중 모바일 회선 회원 및 해당 회원이 초대한 모바일 회선 회원(결합 미이용자 및 타사 모바일 회선 회원 포함)는 이용약관 동의 후 서비스를 이용할 수 있습니다.
단, 일반 가족 회원의 경우, 데이터박스 서비스 이용이 제한됩니다.
서비스 이용 조건
데이터박스 서비스는 결합 가족 회원에게만 제공됩니다.
쿠폰박스 서비스의 경우, 결합 가족 회원 및 일반 가족 회원에게 제공됩니다.
하나의 가족그룹은 하나의 결합그룹으로 구성될 수 있습니다.
복수회선 회원의 경우 모바일 회선별로 서비스 가입을 해야 합니다.
결합 가족 회원의 서비스 탈퇴 또는 결합 해지로 인해 가족 그룹 내 결합 가족 회원이 없는 경우, 본 서비스의 이용이 정지됩니다.
결합 상품의 결합서비스계약번호가 변경된 경우, 기존 결합상품은 해지된 것으로 간주합니다.
본 서비스의 주요 혜택들은 상업적인 목적으로 활용할 수 없으며, 회사는 사용자의 불법/편법적인 서비스 이용 확인시 이용정지, 서비스탈퇴 등 서비스의 이용을 제한할 수 있습니다.
서비스 사용을 위해 이동통신서비스를 이용하는 경우 데이터통화료가 과금됩니다.
본 서비스는 iOS, Android OS를 탑재한 단말기에서 전용앱을 다운로드하여 사용 가능하나, 단말기의 종류와 기능, OS의 버전에 따라 이용이 제한될 수 있습니다.
본 서비스의 이용조건과 혜택 내용은 회사의 서비스 제공 정책에 따라 변경 또는 종료가 가능하며 이 경우 회사는 회원에게 최소 30일 이전에 공지합니다.

보너스 데이터

패밀리박스에 가입하고, KT 결합 상품을 이용 중인 모바일 회선에 아래와 같이 매월 ‘보너스 데이터'를 제공

  • 모바일 보너스 데이터: KT 결합 상품을 이용 중인 결합 가족 회원의 모바일 회선당 100MB 제공
  • 인터넷 보너스 데이터: KT 결합 상품을 이용 중이면서, 결합 상품 내에 인터넷을 이용 중인 경우, 결합 가족 회원의 모바일 회선당 100MB 제공

혜택 제공 기준

  • 매월 1일 0시 기준으로 결합 및 모바일, 인터넷 상품 이용 여부를 확인하여 데이터박스에 제공. 일시정지 및 이용정지 상태인 경우 보너스 데이터 혜택이 제공되지 않음

유의사항

  • 보너스 지급 대상 산정에 요금제 제한은 없으나, ‘데이터 꺼내기 사용 가능 요금제’에 해당하는 모바일 회선만 본인 회선으로 꺼내어 사용이 가능함(데이터 꺼내기 사용 가능 요금제는 데이터 꺼내기 기능 설명 참조)
  • 월 중 신규 가입한 결합 가족 회원 회선의 경우, 보너스 데이터는 가입한 다음날 데이터박스에 지급
  • 지급된 보너스 데이터는 지급된 시점 기준 다음달 말일까지 유지됨
  • 보너스 데이터는 개인 회선으로 꺼내기 한 후 실사용 가능함
  • 당월 보너스 데이터 지급 이력이 있는 회선은 탈퇴 후 재가입, 결합상품 변경 등 어떠한 경우에도 보너스 데이터가 재지급되지 않음
미션 리워드 데이터

서비스 내 미션 수행에 따라 미션별 책정된 한도에 해당하는 리워드 데이터를 지급

  • 미션 리워드 데이터는 데이터 박스에 적립됨
  • 미션 참여는 가족 모두(결합 가족 회원, 일반 가족 회원) 가능하며, 해당 데이터의 꺼내기와 사용은 결합 가족 회원만 가능
  • 미션은 ’출석 체크하기’와 그 외 미션으로 구성됨
  • ‘출석 체크하기’는 가족 그룹당 일 1회 참여 가능하며, 가족 구성원 중 최초 1명이 수행하면, 더 이상 수행할 수 없음
  • 출석 체크하기를 제외한 미션은 회원당 월 1회 참여 가능함
  • 미션 리워드 데이터는 가족 그룹당 월 최대 2,000MB까지 획득 가능하며, 2,000MB에 도달할 경우, 추가적으로 미션을 수행할 수는 있으나 리워드 데이터 지급은 중단됨
  • 미션 내용과 지급 데이터량은 사전 고지 없이 변경될 수 있음
데이터 담기

요금제의 기본제공 데이터 중 원하는 용량을 데이터박스에 적립하는 기능

이용 용량 및 횟수

  • 결합 가족 회원 회선당 월 최대 2,000MB까지 가능, 회당 100MB단위로 2,000MB까지 선택가능
  • 담기 완료 후 기본제공 데이터 잔여량이 500MB 미만인 경우 데이터담기 제한
  • 회선당 데이터 담기/꺼내기 합산 월 30회까지만 이용 가능
  • 한번 담은 데이터는 취소 불가능

보유기간 및 사용 처리

  • 데이터박스에 담은 데이터는 담기 완료 시점 기준 다음달 말일까지 유지됨
  • 데이터박스에 담기 완료한 데이터는 실제 사용한 것으로 간주함

데이터 담기 사용가능요금제

  • 500MB이상의 기본제공 데이터가 제공되는 LTE 및 5G 휴대폰 요금제에서 사용 가능
  • 데이터 무제한 요금제의 경우 기본 제공 공유 데이터 범위 내에서만 가능

이용정지/일시정지 고객의 사용

  • 이용정지/일시정지 고객은 사용 불가
  • 박스에 담은 데이터의 총량이 일시정지로 인한 요금제별 일할 계산된 기본제공 데이터량보다 많은 경우 초과 과금이 발생할 수 있음
데이터 꺼내기

데이터박스에 담긴 데이터를 본인 회선으로 꺼내어 사용가능 상태로 전환하는 기능

  • 본인 회선의 기본 제공 데이터와 구분 되는 '패밀리박스 데이터'로 꺼내어짐 (100MB단위로 이용 가능함)

이용 조건

  • 100MB 단위로 꺼내기 가능하며 회선당 꺼내기 용량의 제한 없음
  • 회선당 데이터 담기/꺼내기 합산 월 30회까지만 이용가능
  • 꺼내온 데이터는 취소 불가능하며 꺼낸 데이터는 다시 담을 수 없음.
  • 꺼낸 데이터는 데이터박스 내 소멸 기간과 상관 없이 꺼내기 시점 기준 당월 말일까지 사용 가능

데이터 꺼내기 차감 순서

  • 데이터박스 내 담긴 시간 순으로 꺼내어지며 담긴 시간이 같을 경우 담은 데이터 또는 모바일 보너스데이터 -> 인터넷 보너스 데이터 -> 이어쓰기 보너스 데이터 ->리워드 데이터 순으로 꺼내어짐

패밀리박스 데이터의 회선 내 차감 순서

  • 꺼내온데이터(패밀리박스데이터)는 기본제공데이터(이월데이터포함)에 우선하여 차감되며 일부 데이터 월정액 부가서비스의 경우 가입 우선 순으로 차감됨(요금제 별 세부 차감 기준은 KT 홈페이지 '개인상품 이용약관'을 참고)
    • - 일반(종량)형: 패밀리박스데이터→ 데이터플러스(가입 우선 순)→ 기본제공(당월)
    • - 이월형: 패밀리박스데이터→ 데이터플러스(가입 우선 순)→이월데이터→기본제공(당월)
    • - 안심차단형: 패밀리박스데이터→기본제공(당월)→충전데이터
    • - 일반비기: 패밀리박스데이터→데이터플러스→별결제 데이터→매직엔알→수신알→기본알→이월알→자동상한알→충전알
    • - 알캡상한: 패밀리박스데이터→데이터플러스→별결제 데이터→매직엔알→알캡상한알

데이터 꺼내기 사용 가능 요금제

  • LTE 및 5G 휴대폰 요금제와 3G요금제 중 청소년 요금제에서 사용 가능 (데이터 이용이 제한된 음성 전용 요금제 제외)
  • 요금제 변경시 잔여 패밀리박스데이터(꺼내온 데이터)는 소멸됨

이용정지/일시정지 고객의 사용

  • 이용정지/일시정지 고객은 사용 불가

KT 결합 구성원간 사용하던 단말을 이어쓸 경우 매월 2,000MB의 추가 데이터를 제공하는 ‘가족폰 이어쓰기’ 서비스를 제공합니다.
동일한 KT 결합 상품을 이용 중인 회선이 KT에서 최근 3년 내 누적 사용일 30일 이상 이용했던 스마트폰에 유심을 삼입하여 개통하고, 패밀리박스 앱을 통해 ‘가족폰 이어쓰기’ 혜택을 신청하면, 매월 2,000MB의 ‘이어쓰기 보너스 데이터’를 데이터박스에 제공합니다. (최대 12개월간)
이어쓰기 보너스 데이터는 최초 신청 월은 당일, 이후 매월 말 이어쓰기 신청 단말의 사용 여부를 검증하여 익월 1일 데이터박스에 제공됩니다. 타 휴대폰으로 유심변경 시 혜택이 지급되지 않습니다.
가족폰 이어쓰기 서비스는 회선당/단말당 1회에 한하여 신청 가능합니다.
쿠폰박스 안내
쿠폰박스 안내
쿠폰박스는 가족이 공용으로 쓸 수 있는 다양한 통신 혜택과 제휴 혜택을 제공합니다.
일부 쿠폰은 적용 대상이 한정적일 수 있으며, 자세한 내용은 쿠폰 상세 정보를 통해 확인해 주시기 바랍니다.
각 쿠폰은 명시된 기간 동안 사용할 수 있으며, 타인에게 임의로 양도하는 행위는 금지합니다.
쿠폰별 유의사항은 각 쿠폰의 상세 정보를 확인하세요.
쿠폰 ‘사용 완료’ 처리는 가족 끼리 사용자/사용 시점 확인을 위한 것이며, 실제 쿠폰 사용여부와는 다를 수 있습니다.

쿠폰박스 주요 기능 별 상세 내용 안내
기능 상세내용
KT WiFi 무료 이용권
  • KT WiFi Zone에 접속 가능한 KT WiFi 무료 이용권을 제공하는 혜택
    • 가족 회원이 신청한 경우 가족 그룹 당 1개의 비밀번호(PIN)를 제공합니다.

이용 조건

  • 비밀번호(PIN)는 WiFi 모듈을 탑재한 1개의 기기에서 사용 가능합니다.
  • 비밀번호(PIN)는 매월 1일부터 신청할 수 있으며 매월 말일 소멸됩니다.(비밀번호(PIN) 유출 방지 목적)
  • 서비스에 1 회선 이상이 유지되는 한 매월 재신청하여 사용할 수 있습니다.
  • 사용 단말 변경을 원할 경우 고객센터에서 본인 인증을 거쳐 네트워크 고유번호(MAC/ID) 변경 후 사용 가능합니다.

기타

  • 제공되는 상품은 KT WiFi싱글 상품에 준하는 상품입니다.
  • 회선 탈퇴로 인해 서비스 가입 결합 가족 회원 회선이 1회선 미만이 되면 자동 소멸됩니다.

패밀리 캘린더 안내
패밀리 캘린더는 가족이 공용으로 쓸 수 있는 공용 캘린더 서비스를 제공합니다.
패밀리 캘린더 서비스는 가족 모두 사용 가능하며, 회원 동의 시 본인의 일정과 기념일 등을 가족과 공유합니다.
