유의사항

[공통]
  • 만 34세 이하 5G 대상 요금제 가입자에게 적용되는 무료 혜택으로 만 35세 생일 익월 1일에 Y덤 혜택이 자동종료됩니다.
  • 가입 조건
    • 1. 5G 일반 요금제: 만 34세 이하 5G 요금제 이용 고객(청소년요금제/Y요금제 등 전용 요금제 제외)
    • 2. 5G 요고 요금제: 만 34세 이하 요고 다이렉트 이용 고객
  • 가입자 1인당 1개 번호에 대해 Y덤 혜택이 적용되며, 5G(Y스페셜, 베이직, 슬림) 및 LTE (Y데이터 ON 프리미엄/비디오플러스/톡플러스) Y요금제와 중복 가입 불가. 또한 다회선 보유 시 A회선이 Y요금제일 경우 B회선은 Y덤 혜택 적용 불가합니다.
  • Y덤 가입 시 요금제 명칭은 ‘요금제명_Y덤‘ 으로 변경됩니다.
[기본 데이터 2배]
  • 5G 일반 요금제는 심플 110GB(69,000원) 월정액 이하, 요고 다이렉트 요금제는 요고 55(55,000원) 월정액 이하 요금제를 이용하는 경우 각 요금제별 기본 데이터의 2배를 사용할 수 있습니다.
  • 월중에 Y덤 혜택이 적용되는 경우 데이터는 일할 계산 되어 제공됩니다.
    (ex. 6.2일 심플 110GB 요금제에 Y덤 혜택 적용되는 경우 당월은 110GB 일할 계산 된 106GB 데이터 추가 사용이 가능)
  • 데이터 차감 순서 : Y덤 제공량 먼저 차감 후 소진 시 요금제에 제공되는 기본 데이터가 차감됩니다.
  • Y덤으로 제공된 기본데이터는 Y박스/패밀리박스에서 데이터 나눔이 불가하며, 데이터 나눔은 요금제에 제공되는 기본 데이터 내에서만 2000MB 한도로 가능합니다.
[스마트기기/테더링 공유데이터 2배]
  • 스마트기기/테더링 공유데이터는 테더링을 이용하거나 스마트기기 요금으로 데이터를 사용하는 경우에 차감되며, Y덤 혜택으로 기존 대비 최대 2배를 사용할 수 있습니다.
  • 스마트기기/테더링 공유데이터는 아래 요금제에 적용되는 혜택입니다.
    • 1.5G 일반 요금제
      • 1) 5G 베이직(80,000원) 월정액 이상 무제한 요금제의 경우 공유 데이터 한도 2배 적용 됩니다.
      • 2) 5G 슬림 21GB(58,000원) 월정액 이하 요금제의 경우 기본데이터 2배 적용되며, 스마트기기/테더링 공유데이터 한도는 요금제별 추가 공유 데이터 한도 내에서 사용 가능합니다.
    • 2.요고 다이렉트 요금제
      • 1)데이터 무제한 요금제: 요고 61(61,000원) 월정액 이상 요금제 공유 데이터 한도 2배 적용 됩니다.
    *차감순서: Y덤 제공량 먼저 차감 후 소진시 기본 요금제의 스마트기기/테더링 데이터가 차감됩니다.
    * Y덤 이월 데이터는 기본 데이터로 사용 가능합니다.
[스마트기기 요금 1회선 무료/할인]
  • Y덤 가입자는 초이스 베이직(90,000원) 요금제 이용 시 스마트기기 요금제 1회선 무료 또는 5G 데이터쉐어링 1회선 무료를 택1할 수 있으며, 베이직(80,000원) 요금제 이용 시 스마트기기 요금제 1회선 50% 할인이 적용 됩니다.
  • 무료/할인 대상 스마트기기(패드/워치) 요금제 : 5G데이터투게더, LTE데이터투게더(데이터투게더Large, 데이터투게더 Medium, 데이터투게더 Watch), Y주니어워치, 키즈80
  • 스마트기기 요금 무료/할인 혜택은 kt 대리점, 고객센터(114)를 통해 별도 신청 시 제공됩니다.
  • 휴대폰 회선과 스마트기기 회선이 동일 명의인 경우 할인이 제공 됩니다. (단, Y주니어워치, 키즈80은 가족명의 회선일 경우에도 할인이 제공 됩니다)
  • Y덤 혜택이 적용된 휴대폰 회선 기준으로 스마트기기 1회선 요금 무료/할인 혜택이 제공됩니다.
  • 휴대폰 요금제 변경 및 정지 시에는 할인을 제공하지 않습니다.
