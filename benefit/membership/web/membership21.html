      이용 고객만 사용할 수 있습니다. (개인, 개인 사업자만 가능하며 선불, 법인은 불가 /

기능 상세내용
데이터 나눔
  • 본인 요금제의 데이터 기본제공량을 차감하여 Y박스 가입자에게 차감한 데이터량만큼 데이터를 양도하는 기능
  • 데이터 나눔 한도
    • 요금제의 당월 데이터 기본제공량 내에서만 나눔 가능(일 제공 데이터 또는 속도제어 데이터가 제공되는 요금제의 경우, 일 제공 데이터와 속도제어 데이터는 나눔 불가)
    • 월 최대 2,000MB 한도 내에서 나눔 가능(데이터 선물하기, 데이턱 합산 기준)
    • 나눔 완료 후 본인 데이터 잔여량이 500MB 이상일 경우에만 가능
    • 나눔 횟수는 제한 없음
  • 2가지 방법 제공
    1. ① 데이터 선물하기
      • 특정 1명을 지정하여 데이터를 나눠주는 기능
      • 회당 100~2,000MB 선물 가능
      • 이용중인 휴대폰의 주소록에 저장되어 있는 전화번호 중 Y박스에 가입한 지인에게만 선물 가능
    2. ② 데이턱
      • 특정인 지정 없이 다수에게(최대 20명) 데이터를 100MB씩 나눠주는 기능
      • 회당 최대 2,000MB 데이턱 가능
      • 데이턱 실행 시 데이터를 받을 수 있는 링크가 자동생성되며 SNS를 통해 지인에게 공유하면 지인들은 링크를 통해 Y박스에 접속하여 선착순으로 데이터를 받을 수 있음
      • 데이턱 링크를 생성한 날을 기준으로 다음 날 밤 12시까지 데이터 받기가 가능하며, 받아가지 않은 데이터가 있는 경우 다음 달 말일 밤 12시까지 Y박스로 회수 가능 (회수 가능한 데이터가 있는 고객은 앱 푸시로 알림)
      • 회수된 데이터의 유효기간은 데이턱 생성시간 기준 다음 달 말일까지 유지
      • 받은 데이터는 앱 내 Y박스에 적립됨
      • 데이턱 링크 당 1회선 1번 100MB 받기 가능 (회선당 동일 데이턱에서 2회, 200MB 받기 불가능)
  • 유의사항
    • 데이터 나눔 후 취소 불가능, 나눔 완료 즉시 사용한 것으로 간주하여 월 중 요금제 변경 시 데이터 초과 과금 발생할 수 있음
    • 최초 1회 SMS 본인인증을 통해 비밀번호를 설정한 뒤, 비밀번호를 입력해야만 나눔 가능
    • 청소년 요금제, 만 18세 이하 고객은 데이터 나눔 불가, 받기만 가능
    • 해당 월에 데이터 나눔을 한 이력이 있는 무제한 요금제 또는 차단형 요금제 이용 고객이 월 중간에 요금제를 변경할 경우 일할 계산하여 초과 과금이 발생할 수 있음
나눔 받은 데이터의
적립과 사용
  • 데이터 나눔을 통해 받은 데이터는 받은 사람의 Y박스에 적립됨
    • 데이터 나눔을 통해 월 최대 2,000MB까지 적립 가능
    • 가입축하데이터 및 보너스데이터의 경우 데이터 나눔 받기 한도에서 제외
    • 적립된 데이터는 적립일 기준 다음 달 말일까지 보관됨 (월 중 이용정지/일시정지를 하더라도 유효기간 연장되지 않음)
  • 받은 데이터 이용을 원할 때 Y박스에서 내 데이터로 꺼내어 이용할 수 있음
    • 회당 100~2,000MB 단위로 꺼내어 이용 가능(월 최대 30회)
    • 꺼낸 데이터는 당월 말일까지 이용 가능
    • 꺼내기 완료 후에는 취소 불가함
    • 꺼낸 데이터 이용 시 ‘데이터 관련 팩 상품 > 데이터 충전 > Y박스 꺼낸 데이터 > 데이터 기본제공량 > 데이터 쿠폰‘ 순으로 차감됨
  • 내 데이터 박스에 보관 중인 데이터, 꺼내어 이용 중인 데이터는 데이터 나눔 불가
  • 꺼낸 데이터를 이용한 음성통화(무선 인터넷 통화(mVoIP)), 데이터 쉐어링은 불가
※ Y틴 프리덤이란?
  • 개요 :
    중고등학생 시험 종료 직후 2주간 Y박스에서 다양한 혜택을 몰아주는 정기 혜택 프로그램
  • 대상 :
    만 12세 ~ 만 18세 청소년 KT 고객
  • 기간 :
    중간/기말고사 직후 2주간, 연 4회 (일반적인 학사일정을 따르며, Y박스 내 별도 공지)
  • 주요 혜택 :
    Y SHOP 할인 / KT결제 할인 / 지니뮤직 이용권 / 기타 10대 타겟 쇼핑 및 식음료 무료 쿠폰 등
    (혜택은 차수에 따라 변경될 수 있으며, 일부 쿠폰은 선착순으로 다운 가능합니다.)
  • 이용 방법 :
    기간 중 Y박스 내 이벤트 페이지에서 혜택 신청 후 Y박스 리워드 메뉴에서 난수번호 쿠폰 확인
    혜택 브랜드에 직접 난수번호 쿠폰을 입력하여 사용 (사용처 및 사용 방법은 쿠폰별 유의사항에 따름)
    *KT결제 할인의 경우 Y박스 쿠폰 다운로드 없이 링크된 별도 페이지에서 신청
2. Y SHOP
Y shop 분류, 상세내용으로 구성된 주요 기능 안내
분류 상세내용
이용
  • Y박스
    • 비로그인 상태에서도 Y Shop 열람이 가능하며, 구매 시 Y박스 로그인이 필요합니다.
    • Y박스는 통신사에 관계없이 회원가입이 가능하며, Y Shop 역시 누구나 가입, 이용할 수 있습니다.(단 개인, 개인사업자에 한함)
고객센터
  • 상담 가능 시간은 평일 10:00 ~ 17:00 이며, 토,일요일 및 공휴일 휴무 입니다.
    (점심시간 12:00 ~ 13:00)
주문/결제
  • 현금결제는 불가하며, 신용카드, 카카오페이 등 [간편결제수단 추가 예정] 결제수단만 이용 가능합니다.
  • KT 멤버십 포인트 고객에 한해 할인을 받을 수 있으며 세부 내용은 Y박스 앱 내에서 확인 가능합니다.
  • 비회원은 상품 주문이 불가능합니다. 회원가입 및 로그인 후 상품 구매가 가능합니다.
배송
  • 배송기간은 결제일로부터 1~3일 내 출고가 이루어지고 있으며, 배송기간은 택배사 상황에 따라 출고 후 평균 1~2일 내 수령가능합니다.
    단 물류 특이사항이 발생되는 경우 추가 지연이 발생될 수 있습니다.
    ※ 배송이 불가하거나, 품절인 경우 별도 별도 연락드릴 수 있습니다.
  • 주문 후 배송지가 다를 경우 한번에 결제하는 것은 불가능합니다.
    배송지에 따라 각각 주문하신 후 결제해 주세요.
    (대량구매의 경우 고객센터로 별도 문의 바랍니다)
  • 해외 배송이 불가하여 현재는 국내 배송건만 가능합니다.
교환/반품
  • 교환 및 반품 문의는 고객센터로 연락 부탁드립니다.
  • 교환./반품이 불가한 사유 안내드립니다.
    • 1) 고객에 의한 사유로 상품 등이 분실 또는 훼손된 경우
      (ex. 본품상자 훼손, 재포장 불가상품 개봉 시)
    • 2) 상품 등의 가치가 현저히 감소한 경우
    • 3) 상품 수령 후 기간이 경과되어 상품 가치가 상실된 경우
  • 교환/반품은 단순변심에 따른 반품의 경우 7일 이내, 제품 하자에 따른 교환/반품의 경우 14일 까지 가능합니다.
  • 교환/반품 택배비는 단순변심인 경우 왕복 배송비 5천원 고객님 부담으로 진행됩니다.
  • 금액이 같은 제품일지라도 제품 특성상 다른 상품으로 교환은 불가합니다.
    제품 초기불량으로 인한 교환 건은 동일 상품으로만 교환 가능합니다.
영수증
  • 영수증 발행은 고객센터로 문의 부탁드립니다.
