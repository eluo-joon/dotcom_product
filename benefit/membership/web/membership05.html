데이터를 내 맘대로 움직인다! 당겨쓰기 하러가기

당겨쓰기가 가능한 요금제를 알려드려요!

  • LTE 데이터 선택
    요금제
  • Y24 요금제

부족하면 당겨쓰고, 남으면 이월하는
데이터 밀당 서비스

데이터 밀당 서비스는 기존 KT에서만 제공하는 데이터 ‘이월＇서비스에
다음 달 데이터를 미리 ‘당겨＇쓰는 서비스가 합쳐진 것입니다.
LTE 데이터 선택/ Y24 요금제 32.8~54.8 가입자라면 무료로 데이터 밀당 서비스를 이용할 수 있습니다.
데이터가 남는 달은 다음 달로 자동 이월되고,
데이터가 부족하면 다음 달 데이터*를 미리 당겨쓸 수 있습니다.
(* 당겨쓰기 별도 신청 필요)

LTE 데이터 선택 요금제 자세히 보기 Y24 요금제 자세히 보기 (만24세 이하 전용)

꼭! 확인하세요

  • 데이터 당겨쓰기는 LTE데이터 선택 요금제 / Y24 요금제 고객님이 다음 달 기본 제공 데이터를 이번 달에 미리 사용하실 수 있는
    서비스 입니다.
  • 데이터 당겨쓰기는 월 최대 20회, 최대 2,000MB까지 가능합니다.
  • 당겨쓰기 한 데이터는 다음 달 1일 00시의 사용량으로 반영되며, 당겨쓰기 한 데이터 중 남은 데이터는 이월되어 다음달 까지
    사용 가능합니다.
  • 데이터의 차감은 당겨쓰기 이월 데이터 > 이월 데이터 > 요금제 기본 제공 데이터 순으로 차감됩니다.
  • 당겨쓰기 이용 고객이 당겨쓰기 불가능한 요금제로 변경하거나, 명의변경, 해지 시에는 당겨쓰기 한 데이터가 일할 없이 즉시
    취소되며(제공량 소멸), 데이터 사용량을 재계산하여 기본 제공량을 초과한 사용량은 요금이 부과될 수 있습니다.
  • 요금제 변경 시 다음 사항을 유의해주시기 바랍니다.

    당겨쓰기 가능한 요금제로 변경 시, 이미 당겨쓰기 한 데이터량이 변경 신청할 요금제의 기본 제공량을 초과할 경우, 초과 데이터에 대한 요금이 다음 달 사용분으로 청구 됩니다.

    당겨쓰기 불가능한 요금제로 변경 시, 이미 당겨쓰기 한 데이터량이 있을 경우, 해당 데이터에 대한 요금이 당월 사용분으로 청구됩니다.

    당겨쓰기 이용 중에는 요금제 예약변경 신청이 제한됩니다.

    당겨쓰기 후 다음달 초에 요금제를 변경할 경우, 요금제 일할 계산에 따라 1일 00시 사용량으로 반영된 당겨쓰기 데이터에 의해 초과 과금될 수 있으니 주의하시기 바랍니다.
